JOURNAL PDFS

One PDF per issue of The Beckford Journal. Naming pattern:

    beckford-journal-{YEAR}-{ISSUE}.pdf

Examples:
    beckford-journal-2024-01.pdf
    beckford-journal-2023-02.pdf

Next to each PDF, add a .json sidecar with issue metadata, e.g.
    beckford-journal-2024-01.pdf
    beckford-journal-2024-01.json        ← { "title": "...", "year": 2024, "summary": "..." }

I'll index these into /data/journal.json for the Journal page.
