HERO / CAROUSEL IMAGES

Drop the home-page hero/carousel images here. Guidelines:

- 3–6 images works best for a carousel.
- Landscape, wide crops (roughly 16:9 or wider) — they are displayed full-width.
- Long edge 2400–3000px; Next.js will generate responsive variants.
- Prefer atmospheric, high-drama images — interiors of Fonthill-era gothic, dramatic portraits of Beckford, evocative landscapes.
- Name them in display order, e.g.:
      01-fonthill-abbey.jpg
      02-beckford-portrait.jpg
