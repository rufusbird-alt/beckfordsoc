GALLERY IMAGES

Drop your Gallery of Pictures images here. Guidelines:

- Formats: .jpg, .jpeg, .png, .webp (Next.js will optimise at build time)
- File naming: lower-case, hyphens, no spaces
    GOOD:  fonthill-abbey-west-front.jpg
    BAD:   Fonthill Abbey West Front.JPG
- Max long edge 3000px — larger is wasted. Next.js serves smaller variants.
- Captions: create a matching .txt file next to each image with caption + credit line, e.g.
      fonthill-abbey-west-front.jpg
      fonthill-abbey-west-front.txt   ← contains the caption

Once uploaded I'll index them into /data/gallery.json for the Gallery page.
